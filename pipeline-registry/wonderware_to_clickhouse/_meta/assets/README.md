# Pipeline Logos

This directory contains logos for the source and destination systems in the Wonderware to ClickHouse pipeline.

## Directory Structure

```
assets/
├── from/          # Source system logo (Wonderware/AVEVA)
│   ├── logo.png
│   └── README.md
└── to/            # Destination system logo (ClickHouse)
    └── logo.png
```
