# OpenWeather Connector

Access current weather, forecasts, geocoding, and air pollution data from OpenWeatherMap.

## Overview

The OpenWeather connector provides a simple interface to the OpenWeatherMap API, allowing you to:

- Get current weather conditions for any location
- Retrieve 5-day weather forecasts with 3-hour intervals
- Convert between city names and geographic coordinates
- Access air quality and pollution data

## Features

- **Current Weather**: Real-time weather data for 200,000+ cities
- **5-Day Forecast**: Weather predictions with 3-hour granularity
- **Geocoding**: Direct and reverse geocoding, zip code lookup
- **Air Pollution**: Current, forecast, and historical air quality data

## Versions

- **v1**: Initial implementation with core weather APIs
