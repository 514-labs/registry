# Changelog

All notable changes to this connector will be documented in this file.

## 0.1.0 - Initial Release

### Added
- Current weather data retrieval for any location
- 5-day weather forecast with 3-hour intervals
- Geocoding API support (direct, reverse, and zip code lookup)
- Air pollution data (current, forecast, and historical)
- Comprehensive TypeScript type definitions
- Support for multiple units (standard, metric, imperial)
- Multi-language support for weather descriptions
- Built-in logging and metrics capabilities
- Full test coverage with unit tests
- Complete documentation and examples

### Features
- Weather resource for current conditions
- Forecast resource for 5-day predictions
- Geocoding resource for location resolution
- Air pollution resource for air quality data
- Configurable API key authentication
- JSON schemas for all API responses
