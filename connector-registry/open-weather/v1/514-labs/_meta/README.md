# OpenWeather Connector (by 514-labs)

TypeScript connector for the OpenWeatherMap API providing access to weather data, forecasts, geocoding, and air quality information.

## Connector Information

- **Name**: OpenWeather
- **Identifier**: `open-weather`
- **Author**: `514-labs`
- **Category**: API Connector
- **Version**: v1
- **Languages**: TypeScript

## Available Implementations

### TypeScript (default)
Full-featured TypeScript implementation with:
- Type-safe API methods
- Comprehensive error handling
- Built-in logging and metrics
- Extensive test coverage

Path: `typescript/default/`

## API Coverage

### Weather Data
- Current weather conditions
- 5-day forecast with 3-hour intervals

### Geocoding
- Direct geocoding (city name → coordinates)
- Reverse geocoding (coordinates → city name)
- Zip code lookup

### Air Quality
- Current air pollution data
- Air pollution forecast
- Historical air pollution data

## Getting Started

1. Sign up for a free API key at [OpenWeatherMap](https://openweathermap.org/api)
2. Install the connector in your project
3. Initialize with your API key
4. Start making requests

See implementation folders for language-specific documentation and examples.
