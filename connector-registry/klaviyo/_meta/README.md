# Klaviyo (Registry)

Klaviyo is a marketing automation and CRM platform that enables businesses to manage customer relationships, send targeted email campaigns, and track customer behavior.

## Overview

The Klaviyo connector provides access to:
- **Profiles**: Customer/contact data with email, phone, and custom properties
- **Lists**: Audience segments and subscription lists
- **Campaigns**: Email marketing campaign information

## Available Versions

- `v1`: Latest API implementation (revision 2024-10-15)

## Implementations

- `klaviyo/v1/514-labs/typescript/default`: TypeScript implementation by 514-labs
